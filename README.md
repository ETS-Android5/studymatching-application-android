# 스터디 매칭 애플리케이션 개발
## 스터디가 필요한 사람들을 한 곳으로 모으는 애플리케이션
> 배경 : 
> > - 스터디를 찾는 방대한 커뮤니티로 인한 사용자 혼란 가중
> > - 다양한 공부 분야로 searching or matching에 어려움
> 목적 :
> > - 플랫폼 통일
> > - 다양한 스터디에 대한 정보를 한 눈에 파악할 수 있음
> 기능 :
> > - 회원가입 기능
> > - 로그인 기능
> > - 마이페이지 기능
> > - 비밀번호 찾기 기능
> > - 스터디 생성/입장
> > - 커뮤니티 기능
> > - 공지사항 기능
> > - 실시간 채팅 기능
> > - 위치 기반 도서관 검색 기능
> > - 화상회의(Zoom) 지원
